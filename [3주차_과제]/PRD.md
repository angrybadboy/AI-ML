## 1. 개요

| 항목 | 내용 |
|------|------|
| 제품명 | 신경망 인터랙티브 트레이너 |
| 버전 | 1.0 |

| 목적 | Week 3 스스로 해보기 5개 항목을 GUI 환경에서 실험 |

---

## 2. 배경 및 목표

Week 3 수업에서 Multi-Layer Perceptron(MLP)의 이론을 학습했다.
스스로 해보기 항목은 하이퍼파라미터(뉴런 수, 학습률, 활성화 함수)와 네트워크 구조(2-layer / 3-layer)가 학습 결과에 미치는 영향을 직접 관찰하는 것이다.

매번 코드를 수정하고 재실행하는 방식은 비효율적이므로, **GUI 앱으로 파라미터를 실시간으로 바꿔가며 결과를 즉시 비교**할 수 있는 도구를 만든다.

---

## 3. 사용자

- 대상: AI/ML 입문 수강생 (본인)
- 환경: macOS, Python 3.12+, PySide6

---

## 4. 기능 요구사항

### 4.1 핵심 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 은닉층 뉴런 수 선택 | 은닉층 1: 2 / 4 / 8 / 16 중 선택 |
| F-02 | 학습률 선택 | 0.01 / 0.1 / 0.5 / 1.0 중 선택 |
| F-03 | 활성화 함수 선택 | Sigmoid / ReLU / Tanh 중 선택 |
| F-04 | 3-layer 네트워크 | 은닉층 2 뉴런 수(없음/2/4/8/16) 선택으로 2-layer ↔ 3-layer 전환 |
| F-05 | 다양한 데이터셋 | XOR / Circle / Spiral 중 선택 |

### 4.2 시각화 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| V-01 | Loss 곡선 | 학습 epoch별 MSE Loss (log 스케일) |
| V-02 | 결정 경계 | 2D 입력 공간의 출력 확률 컨투어 플롯 |
| V-03 | 은닉층 활성화 히트맵 | 은닉층 1의 뉴런별 활성화 값 |

### 4.3 UX 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| U-01 | 진행 상황 표시 | 프로그레스 바 + 실시간 Loss 값 표시 |
| U-02 | 결과 요약 | 최종 Loss, 정확도, 레이어 구조 표시 |
| U-03 | 비동기 학습 | 학습 중에도 UI 응답 유지 (QThread 활용) |

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 응답성 | 학습 시작 후 UI가 멈추지 않아야 함 |
| 재현성 | 같은 설정으로 재실행 시 유사한 경향의 결과 |
| 의존성 | PySide6, NumPy, Matplotlib만 사용 (외부 ML 프레임워크 없음) |

## 6. 스스로 해보기 항목 대응 매핑

| 과제 항목 | 대응 기능 |
|-----------|----------|
| 1. 은닉층 뉴런 수 바꿔보기 (2, 8, 16) | F-01: 은닉층 1 콤보박스 |
| 2. 학습률 조정 (0.1, 0.5, 1.0) | F-02: 학습률 콤보박스 |
| 3. 다른 활성화 함수 시도 | F-03: 활성화 함수 콤보박스 |
| 4. 3-layer 네트워크 구현 | F-04: 은닉층 2 콤보박스 |
| 5. 다른 데이터셋 시도 | F-05: 데이터셋 콤보박스 |

## 8. 성공 기준

- XOR 문제에서 정확도 100% 달성 가능
- 학습률 1.0에서 발산(Loss 급증) 현상 관찰 가능
- 활성화 함수 변경 시 수렴 속도 차이 시각적으로 확인 가능
- 3-layer가 2-layer보다 복잡한 패턴(Spiral) 학습에 유리함 확인 가능
